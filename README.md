# 📚 Booky – Book Tracker (Frontend)

A modern book tracking web application that helps users manage their reading journey.

Booky allows users to:
- Add books
- Track reading progress
- Categorize books (To Read, In Progress, Completed)
- View reading statistics
- Manage notes

---

## 🚀 Tech Stack

- ⚛️ React
- ⚡ Vite
- 🎨 Tailwind CSS
- 🧩 ShadCN UI
- 📊 Radix UI
- 🔐 Clerk Authentication (if used)
- 🌐 Axios (API calls)

## ✨ Features

### 📈 Progress Tracking
